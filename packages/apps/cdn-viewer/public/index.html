iTwin Viewer

To consume the iTwin Viewer in your application, add a script tag whose src is this url plus vX/static/js/web-viewer-react.js (i.e. itwinviewer.bentley.com/v1/static/js/web-viewer-react.js) and a style tag whose href is this url plus vX/static/css/web-viewer-react.css (i.e. itwinviewer.bentley.com/v1/static/css/web-viewer-react.css). bundle.
